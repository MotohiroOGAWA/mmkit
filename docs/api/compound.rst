Compound
========

Represents a chemical compound as an RDKit molecule with canonical SMILES,
molecular formula, charge, exact mass, and atom mapping support.

The :class:`~mmkit.Compound.Compound` class provides functionality for creating,
canonicalizing, and inspecting chemical structures. It supports construction
from SMILES or RDKit molecules, retrieval of molecular properties such as
formula and exact mass, and handling of atom map numbers for structure-aware
workflows in cheminformatics and mass spectrometry.
